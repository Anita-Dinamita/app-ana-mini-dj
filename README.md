# Ana Mini DJ

Sitio web estático de **Ana Mini DJ** — una única página HTML autocontenida (`index.html`).

## Despliegue

- **GitHub:** repositorio fuente.
- **Vercel:** desplegado como sitio estático con auto-deploy en cada push a `main`.

No requiere build ni dependencias: es HTML + CSS embebido (fuentes servidas desde Google Fonts).
